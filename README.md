# Working-at-SHIC
## work
1. 要有足够高的起点，要不断进步，每一次的进步要成为下一次进步的基点、而不是重新开始。要做到高速率、可叠加式的进步，要懂得放弃，放弃所有不能对长远发展有用的短期机会。
2. 自觉，就是眼里有活。
3. 主动让上级了解自己的工作。
4. 分享利益、感激他人的贡献，是基本原则。
5. 不论别人是否有错误，现在你错了就该被惩罚。
6. 标榜自己是最会写程序的产品经理，那只能说明是半瓶子醋。我们要做的是超过他人的长处，而不是满足于超越别人的短处。
7. 功夫没下够，用什么方法都是浪费时间。先动脑再动手，把可做可不做的事情从任务清单删除，在做事的过程中把事情按部就班做好。不要只开头不结尾，做完要进行评价。
8. 巴菲特摇摇头:从现在起，你要把精力集中在这五个目标上，再也不要去想剩下的那20个目标，那些都是你人生道路上的陷阱。 驾驶员跟很多人一样，听到这句话的时候很懵，然后他们不得不承认，巴菲特这句话是对的。一个人的一生，能完成五个目标，就已经是很了不起的事了，而把精力放在更多的目标上，只会让自己掉到人生的陷阱之中。
9. 如何挑选股票？找一个对投资人好的公司。找工作时，找一个对自己好、能帮助自己成长的公司。
10. 关注是不值钱的，每用户收益值才是关键。买关注很容易，加名人微信很容易，远不如提升arpu值。优质的才是稀缺的。最珍贵的是时间，不要花时间读免费廉价的东西。
11. 能维持朋友关系的只有几十个。不给骗人的人第二次机会。
12. 牙刷与爆款：增高曝光率，偶尔给一次惊喜。
13. 降低生活标准去买房买车是不值得的。不要通过延长工作时间挣钱。
14. 贵金属不是好的投资。
15. 理财顾问并没有你想的那么聪明。
16. 培養健康的習慣， 觀察周遭人有哪些好的工作習慣或行為模式，同時鼓勵自己做出改變，逐步培養這些良好的工作習慣。
## politics
   如果你希望在组织中能更好地运用政治技能，请看如下八条建议。
1. 把观点包装成是为了组织的目标。有效的政治活动要求你必须掩饰你的私人利益。不管你的最终目的是不是为了自己，你所主张的所有观点最终都是为了组织的利益服务的。公然表现出牺牲组织而为自己谋利的人几乎最终都会受到谴责，失去影响力，而且通常会被组织开除。
2. 建立正确的形象。如果你了解组织的文化，你知道组织想要实现什么、想要员工做什么（包括着装规定、应建立哪些联系而避免哪些、风险爱好还是风险规避、偏好的领导风格、与他人和睦相处的重要性），你就可以据此为自己建立一个良好的形象。因为你对自己的业绩评估并不是完全客观的，所以你需要注意自己的风格和能力。另外，研究表明成功地表现真诚的人通常会给人一个正面的形象。
3. 得到组织资源的控制权。对组织稀缺且是重要资源的控制是权力的来源。知识以及经验是格外有效的资源，它们能让你对组织更有价值，从而是你更加安全和进步，为你的想法赢得支持。
4. 让你看起来不可或缺。因为我们更多面对的是表象而不是客观事实，你可以通过“看起来不可或缺”来增强你的权力。你并不用真的那么厉害，只要组织中关键的人物相信你是不可或缺的就可以了。如果组织的首要决策者相信没什么可以代替你为组织做出的贡献，他们就会更大限度地保证满足你的愿望。
5. 保持可视性。如果你的工作能让别人注意到你的成就，那将是不错的。然而，如果你没有这样一份工作，你就得想办法让组织里其他人知道你在做什么，比如在常规报告里强调自己的成绩，将高满意度的顾客反馈和赞赏转达给高管，出席社交场合，活跃在专业协会组织中，结交能为你的成就做出积极评价的强大盟友。当然，有技巧的政治家们活跃而成功地到处游说，可以为自己赢得增加曝光率的机会。
6. 发展强大的盟友。如果你身边存在有权势的人，对你将是很大的帮助。通过联系有潜在影响力的人建立起社交网络，不管他的社会地位和你一样还是高于或低于你。这些盟友能给你提供其他途径得不到的消息。此外，决策有时是按照多数支持原则制定的，在你需要的时候，强大的盟友能给你提供强大的支持。
7. 远离有污点的人。几乎在每个组织中都会有少数地位有争议的人，他们的业绩和忠诚度都很可疑。与这些人保持距离吧，考虑到有效性很大程度是由主观因素决定的，如果你与这些人走很近，你自己的工作也会受到怀疑。
8. 支持你的上司。你现在的上司决定着你职业下一步的走向。这个人负责评估你的业绩表现，所以你做的一切必须得到他的支持。你必须不遗余力地帮助他成功，让他看起来光彩照人，在他被孤立的时候支持他，并且努力发现他考察你工作有效性的标准。不要低估你的上司，也不要和别人说他的坏话。

# 公司希望招聘到什么样的学生
第一印象(first impression) :知觉主体与陌生人第一次接触或交往后的所得印象。
第一印象主要是根据对方的表情、姿态、身体、仪表和服装等形成的印象。第一印象在日常生活中是很普遍的，这种初次获得的印象往往是今后交往的依据。
人与人第一次交往中给人留下的印象，在对方的头脑中形成并占据着主导地位，这种效应即为第一印象效应。也叫首次效应，首次效应的影响会大于后面时间所产生的影响。
