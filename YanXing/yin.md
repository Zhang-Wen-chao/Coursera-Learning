# 公司希望招聘到什么样的学生
第一印象( first impression） :知觉主体与陌生人第一次接触或交往后的所得印象。
第一印象主要是根据对方的表情、姿态、身体、仪表和服装等形成的印象。第一印象在日常生活中是很普遍的，这种初次获得的印象往往是今后交往的依据。
人与人第一次交往中给人留下的印象，在对方的头脑中形成并占据着主导地位，这种效应即为第一印象效应。也叫首次效应，首次效应的影响会大于后面时间所产生的影响。
## 企业里最受不了的员工是这样的对待工作的态度:
1. 没有上进心，不愿意为工作付出更多努力但对各种薅羊毛的活动分外积极，摆烂;
2. 混时间，无清晰规划和明确目标;
3. 不主动承担工作任务;
4. 做事前爱讲条件，工作配合度差。
## 企业里最受不了的员工是这样的工作时的表现:
1. 工作不积极，闷葫芦，工作永远等指示;
2. 主观能动性不足，需要不停督促;
3. 执行力低，做事拖拉;
4. 工作不认真，没有责任心，没有担当;
5. 不善于思考，不积极改善和创新;
6. 做事只执行命令不思考，不主动,推一步走一步。
## 企业里最受不了的员工是这样的工作时的表现:
1. 不主动沟通，不了解计划或任务就去执行，过程没有及时反馈和沟通;
2. 遇事没有自己的思考，啥事儿都来问;
3. 工作爱耍小聪明，工作效率低;
4. 当面沟通达成一致,但行动及成果与当时沟通相差甚远;
5. 推卸责任，爱甩锅;
6. 目标未完成，不会从自身反思。
## 企业里最受不了的员工是这样的团队合作:
1. 无团队意识，挑同事毛病，越级汇报;
2. 和同事关系搞得太差;
3． 在团队内部传递负能量;
4. 打小报告背后使坏;
5. 背后对同事说三道四，当面客客气气;
6. 弄虚作假，没能力也不配合，搞虚头巴脑的事。

## 进入公司需要注意什么
### 进公司第一件事
认识同事、认识领导，和大家建立信任，融入团队。这需要:
诚信道德、遵纪守法懂得感恩、尊重他人思维正向、充满热情团队合作、积极主动踏实肯干、脚踏实地有沟通能力、有同理心

### 进公司第二件事
学习公司文化、制度、流程、工作要求、工作方法、相关的法规法律要求等等。这需要:
能吃苦、爱学习、会学习、
有自我管理能力、
有领导力

### 进公司第三件事
能够按时保质保量的完成工作。这需要:
